# Smiley Bank APP

## Overview
Smiley Bank is a web application that allows teachers to track student points in a fun and visual way. Each student has a “piggy bank” that stores their points, making it easy to reward positive behaviour and monitor progress over time.

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLAlchemy (database)

## How To Run The App

1. Install Python from:
   https://www.python.org/downloads/

   IMPORTANT: During install, tick "Add Python to PATH"

2. Open this folder

3. Double click "run.bat"

4. The browser will open the Smiley Bank!
